# Nova Sondagem

Micro front-end para a nova sondagem, este sistema é integrado ao SGP.

## Instalação

Instale as dependencias com o npm (requer node 16)

```bash
  npm install
```

Para executar o projeto execute o comando

```bash
  npm mf:dev
```

Acessar o menu "Diário de classe" -> "Sondagem" do SGP para visualizar a interface desse projeto
